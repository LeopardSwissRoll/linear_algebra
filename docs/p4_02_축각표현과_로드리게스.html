축-각 표현 & 로드리게스

임의의 축 + 각도 하나로 3D 회전. 로드리게스 공식으로 직접 회전한다.

회전의 자유도 — SO(3)

어떤 3D 회전이든 축 하나 + 각도 하나로 표현 가능 (오일러의 회전 정리)

축 방향 2자유도 + 각도 1자유도 = 총 3자유도. SO(3) = 직교행렬 중 $\det=+1$.

회전 축 $\hat{n}$과 각도 $\theta$. 축 위의 점은 움직이지 않는다 — 축 = 고유벡터.

축-각

축 $\hat{n}$
각도 θ

로드리게스 회전 공식

$$\vec{v'} = \vec{v}\cos\theta + (\hat{n}\times\vec{v})\sin\theta + \hat{n}(\hat{n}\cdot\vec{v})(1-\cos\theta)$$

평행 성분은 불변, 수직 성분만 회전. $\vec{v}_\perp$와 $\hat{n}\times\vec{v}$가 수직면의 직교 기저.

θ60°

분해

$\vec{v}_\parallel = \hat{n}(\hat{n}\cdot\vec{v})$ (축 사영, 불변)
$\vec{v}_\perp = \vec{v} - \vec{v}_\parallel$ (수직 성분)
$\hat{n}\times\vec{v}$ (수직면 직교 벡터)
결과 $\vec{v'}$

축-각의 한계

합성이 불편

두 축-각의 합성 = 직접 공식 없음. 행렬로 변환 → 곱 → 다시 추출.

보간이 비선형

축과 각도를 각각 Lerp하면 결과가 이상해짐.

θ=0의 특이점

회전 없음일 때 축이 정의되지 않음.

→ 쿼터니언이 이 모든 걸 해결한다

합성 = 쿼터니언 곱. 보간 = Slerp. 특이점 없음. 다음 수업에서.

직접 해보기

  1. 축-각 체감: "y축 90°" 프리셋을 눌러봐. Yaw 90°와 같은 결과인지.
  2. 분해 확인: 로드리게스 캔버스에서 축 방향 벡터를 회전. $\vec{v}_\parallel = \vec{v}$이고 결과 = 원래.
  3. 90° 회전: θ=90°에서 $\vec{v}_\perp$가 $\hat{n}\times\vec{v}$ 방향으로 완전히 이동하는 걸 확인.
  4. 행렬 비교: x축 90° 축-각의 결과가 $R_x(90°)$ 행렬과 같은지.
