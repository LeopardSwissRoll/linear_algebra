$qvq^*$ 회전 증명

왜 양쪽에서 곱해야 하는가? 2D 복소수에서 3D로 확장.

복습: 2D에서는 한쪽 곱으로 충분했다

p2_02에서: $z' = e^{i\theta} \cdot z$. 한쪽 곱 한 번이면 끝.

2D에서 양쪽 곱(샌드위치)을 시도하면?

$w = e^{i\theta/2}$로 $wz\bar{w} = e^{i\theta/2} \cdot z \cdot e^{-i\theta/2}$

복소수는 교환 가능 → $z$와 $e^{-i\theta/2}$를 교환 → $e^{i\theta/2} \cdot e^{-i\theta/2} \cdot z = 1 \cdot z = z$

상쇄돼서 항등! 2D에서 양쪽 곱은 무의미. 하지만 3D 쿼터니언은 비교환 → 상쇄 안 됨 → 회전이 작동.

θ60°

3D에서 한쪽 곱이 안 되는 이유

3D 벡터를 순수 쿼터니언으로: $v = (0, \vec{v})$. 단위 쿼터니언 $q = (s, \vec{u})$와 곱하면:

$$qv = (-\vec{u}\cdot\vec{v},\;\; s\vec{v} + \vec{u}\times\vec{v})$$

실수부 $= -\vec{u}\cdot\vec{v} \neq 0$ → 순수 쿼터니언이 아님 = 3D 벡터로 해석 불가!

한쪽만 곱하면 3D 벡터 공간에서 벗어난다. $qvq^*$는 실수부를 다시 0으로 만든다.

증명: $qvq^*$가 회전인 3가지 이유

① 순수 쿼터니언 보존

$v$가 순수이면 $v^* = -v$. $(qvq^*)^* = q v^* q^* = q(-v)q^* = -(qvq^*)$.

$p^* = -p$이면 $p$는 순수. ∴ $qvq^*$는 순수 쿼터니언 = 3D 벡터. ✓

② 크기 보존

$|qvq^*| = |q|\cdot|v|\cdot|q^*| = 1 \cdot |v| \cdot 1 = |v|$. ✓

③ 기하학 = 로드리게스

$v$를 축에 평행/수직으로 분해: $v = v_\parallel + v_\perp$

평행: $\hat{n}$과 $v_\parallel$은 같은 방향 → 교환 → $qv_\parallel q^* = v_\parallel$ (불변)

수직: $\hat{n}v_\perp = -v_\perp\hat{n}$ (반교환) 이용.

왜 수직이면 반교환? 순수 쿼터니언 곱에서 $pq = (-\vec{p}\cdot\vec{q},\;\vec{p}\times\vec{q})$, $qp = (-\vec{q}\cdot\vec{p},\;\vec{q}\times\vec{p})$. 수직이면 내적=0(실수부 동일), 외적만 남는데 $\vec{p}\times\vec{q} = -\vec{q}\times\vec{p}$ → $pq = -qp$.

→ $qv_\perp q^* = v_\perp\cos\theta + (\hat{n}\times v_\perp)\sin\theta$ = 로드리게스 수직 성분 회전. ✓

$$qvq^* = \text{축 } \hat{n} \text{ 중심 } \theta \text{ 회전} = \text{로드리게스 공식}$$

왜 $\theta/2$? — 이중 피복

양쪽에서 각각 절반씩: $q$가 $\theta/2$, $q^*$가 나머지 $\theta/2$ → 합쳐서 $\theta$.

$q$와 $-q$는 같은 회전: $qvq^* = (-q)v(-q)^*$

하나의 회전에 두 개의 쿼터니언 대응 → $S^3$이 SO(3)를 2:1로 피복.

$qvq^*$ 체감

$qv$ (중간 결과)의 실수부 ≠ 0을 확인하고, $qvq^*$에서 실수부 = 0으로 돌아오는 걸 봐.

θ90°

중간 과정

q (단위 쿼터니언)
v (순수: 실수부=0)
qv (중간 — 실수부 ≠ 0!)
qvq* (최종 — 실수부 = 0 ✓)
크기 보존

합성 = 곱셈, 역 = 켤레

$q_2(q_1 v q_1^*)q_2^* = (q_2 q_1)v(q_2 q_1)^*$ → 합성 = 쿼터니언 곱

역회전 = $q^*$. 행렬 곱(27번 연산) 대비 쿼터니언 곱(16번) 저렴.

직접 해보기

  1. 2D 확인: 캔버스 ①에서 $e^{i\theta}z$와 $wz\bar{w}$가 항상 겹치는지.
  2. 실수부: 캔버스 ②에서 $qv$의 실수부(빨강)가 0이 아닌 걸 확인. $qvq^*$에서 0이 되는 걸 확인.
  3. 이중 피복: "−q" 버튼을 눌러봐. 결과가 같은지.
  4. 크기 보존: $|v|$와 $|qvq^*|$가 같은지 패널에서 확인.
  5. 왜 양쪽 곱: "한쪽만 곱하면 3D 벡터 공간에서 벗어난다"를 한 줄로.
