역행렬 & 행렬식

행렬식 = 넓이 변화율, 역행렬 = 되감기. det = 0이면 정보가 사라진다.

행렬식 — 넓이 변화율

$$\det \begin{bmatrix} a & b \\ c & d \end{bmatrix} = ad - bc$$

$|\det|$ = 넓이 배율. 부호 = 방향 보존(+) 또는 뒤집힘(−). 0 = 차원 축소.

det의 곱셈 성질

$$\det(AB) = \det A \cdot \det B$$

A가 넓이를 3배, B가 2배 → AB는 6배. 기하학적으로 당연.

성질이유
$\det(A^{-1}) = 1/\det A$되감으면 원래 넓이로
$\det R = 1$ (회전)넓이 보존
$\det F = -1$ (반사)넓이 보존 + 뒤집힘
$\det(sI) = s^2$ (균등 스케일)변 $s$배 → 넓이 $s^2$배

역행렬 — 변환을 되감는다

$$M^{-1} = \frac{1}{\det M} \begin{bmatrix} d & -b \\ -c & a \end{bmatrix}$$

$\frac{1}{\det}$ = 넓이 되돌림. $a \leftrightarrow d$ 교환, $b,c$ 부호 반전 = 방향 되돌림.

2D → 1D로 찌그러뜨린 변환은 되돌릴 수 없다. 직선 위의 점이 원래 2D의 어디에 있었는지 알 수 없으니까 — 여러 입력이 같은 출력으로 갔으므로 원래 위치를 특정 불가. 이것이 "정보 소실"이다.

차원 축소 — det = 0의 세계

$$\det M = 0 \iff \text{열벡터가 일직선 (선형 종속)}$$

2D 공간이 1D 직선 또는 0D 점으로 붕괴. 클릭해서 "원래 어디에 있었나" 확인.

연립방정식과의 연결

$M\vec{x} = \vec{b}$ → "변환 M을 거쳤더니 b가 나왔다. 원래 x는?"

$\det \neq 0$: $\vec{x} = M^{-1}\vec{b}$ (유일한 해).   $\det = 0$: 해 없음 또는 무한 해.

연립방정식의 "0 = 0" vs "0 = 5"

det = 0일 때, $\vec{b}$가 축소된 공간 위에 있으면 무한 해 ("0 = 0"), 밖에 있으면 해 없음 ("0 = 5").
