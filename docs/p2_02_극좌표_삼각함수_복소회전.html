극좌표·삼각함수·복소 회전

회전 행렬이 왜 저 모양인지, 복소수 곱셈이 왜 회전인지

극좌표계 — 거리와 각도로 위치를 말한다

직교 좌표 $(x, y)$는 "오른쪽으로 $x$, 위로 $y$" — 격자 사고. 극좌표 $(r, \theta)$는 "원점에서 $r$만큼, $\theta$ 방향으로" — 레이더 사고.

$$x = r\cos\theta, \quad y = r\sin\theta$$ $$r = \sqrt{x^2 + y^2}, \quad \theta = \text{atan2}(y, x)$$

삼각함수 — 단위원 위의 좌표

단위원($r = 1$) 위의 점 = $(\cos\theta, \sin\theta)$. 삼각함수는 "각도를 좌표로 바꾸는 함수"다.

$$\sin^2\theta + \cos^2\theta = 1$$

증명이 아니라 정의에서 바로 나온다 — 단위원 위의 점이면 $x^2 + y^2 = 1$이니까.

피타고라스 항등식

덧셈정리 → 회전 행렬

핵심 공식 — 덧셈정리

$$\cos(\alpha + \beta) = \cos\alpha\cos\beta - \sin\alpha\sin\beta$$ $$\sin(\alpha + \beta) = \sin\alpha\cos\beta + \cos\alpha\sin\beta$$

이게 왜 중요하냐? "각도를 더한다 = 회전이다"이기 때문.

유도

점 $P = (r\cos\alpha,\; r\sin\alpha)$를 $\beta$만큼 회전하면, 새 각도가 $\alpha + \beta$이므로:

$$x' = r\cos(\alpha+\beta) = x\cos\beta - y\sin\beta$$ $$y' = r\sin(\alpha+\beta) = x\sin\beta + y\cos\beta$$

행렬로 정리:

$$R(\beta) = \begin{bmatrix} \cos\beta & -\sin\beta \\ \sin\beta & \cos\beta \end{bmatrix}$$

외우는 게 아니라, 덧셈정리에서 자연스럽게 나온다.

β = 60°

회전 행렬의 성질

열벡터가 직교 + 길이 1 → 직교 행렬. $\det R = \cos^2\beta + \sin^2\beta = 1$ → 넓이·방향 모두 보존. $R^{-1} = R^T$ → 역행렬 = 전치. 역회전 = $R(-\beta)$.

복소 평면 — 곱셈이 곧 회전

복소수의 극형식

$$z = x + iy = r(\cos\theta + i\sin\theta) = re^{i\theta}$$

복소 평면에서 실수축 = $x$, 허수축 = $y$. 복소수 하나 = 2D 점 하나.

곱셈의 기하학

$$z_1 \cdot z_2 = r_1 r_2 \cdot e^{i(\theta_1 + \theta_2)}$$

크기는 곱해지고, 각도는 더해진다. $|z_2| = 1$이면 순수한 회전!

복소수는 2D에서만 작동한다. 3D 회전은? 복소수를 확장한 쿼터니언 $q = w + xi + yj + zk$가 필요하다. 복소수에서 "$i$를 곱하면 90° 회전"이듯, 쿼터니언에서는 "$q$를 곱하면 임의의 3D 축 회전"이 된다.

직접 해보기

  1. 극좌표 읽기: 극좌표 탐색기에서 점 (3, 4)를 찾아봐. $r$과 $\theta$는?
  2. 회전 확인: 회전 캔버스에서 벡터 (1, 0)을 90°만큼 회전. 결과가 (0, 1)인지 확인.
  3. 회전 행렬 유도: 벡터 (3, 1)을 45°만큼 회전한 결과를, 행렬 곱으로 직접 계산하고 캔버스와 비교.
  4. 복소수 곱셈: $z_1 = 1+i$, $z_2 = i$일 때 $z_1 z_2$는? 캔버스에서 확인하고, "$i$를 곱하면 90° 회전"이 맞는지 체감.
  5. 회전의 합성: 복소수 곱셈기에서 $z_2$를 단위원 위에 놓고(크기≈1), 여러 각도로 바꿔봐. "곱셈 = 회전"을 체감.
