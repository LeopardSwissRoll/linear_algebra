쿼터니언 대수

$i^2 = j^2 = k^2 = ijk = -1$ — 이 하나에서 모든 곱셈 규칙이 나온다

복소수에서 쿼터니언으로

복소수 (p2_02)

$z = a + bi$, $i^2 = -1$. 기저 2개 $\{1, i\}$. 2D 회전.

쿼터니언 (해밀턴, 1843)

$q = w + xi + yj + zk$. 기저 4개 $\{1, i, j, k\}$. 3D 회전.

기본 정리 — 곱셈표 유도

$$i^2 = j^2 = k^2 = ijk = -1$$

이 하나의 식에서 모든 곱셈 규칙이 나온다.

유도

$ijk = -1$의 양변에 $k$를 오른쪽 곱: $ij(kk) = -k \implies ij(-1) = -k \implies ij = k$

$ijk = -1$의 양변에 $i$를 왼쪽 곱: $(ii)jk = -i \implies -jk = -i \implies jk = i$

같은 방식으로: $ki = j$. 역순은 부호 반전: $ji = -k$, $kj = -i$, $ik = -j$.

곱셈표

비교환! $ij = k$ 하지만 $ji = -k$

순환 규칙: $i \to j \to k \to i$ 순서면 양(+), 역순이면 음(−). 외적 $\vec{e_x} \times \vec{e_y} = \vec{e_z}$와 같은 패턴!

쿼터니언 곱셈

스칼라-벡터 표기: $q = (s, \vec{v})$

$$q_1 q_2 = (s_1 s_2 - \vec{v_1}\cdot\vec{v_2},\;\; s_1\vec{v_2} + s_2\vec{v_1} + \vec{v_1}\times\vec{v_2})$$

외적이 들어가니까 비교환! $\vec{v_1}\times\vec{v_2} = -\vec{v_2}\times\vec{v_1}$

곱셈 계산기

q₁ =
q₂ =
q₁ · q₂
q₂ · q₁

켤레, 노름, 역

켤레: $q^* = (w, -x, -y, -z)$   노름: $|q| = \sqrt{w^2+x^2+y^2+z^2}$   역: $q^{-1} = q^*/|q|^2$

단위 쿼터니언 ($|q|=1$)이면 $q^{-1} = q^*$. 역 = 켤레!

$qq^* = |q|^2$ (실수)

복소수 $z\bar{z} = |z|^2$와 동일.

$|q_1 q_2| = |q_1||q_2|$

노름의 곱 = 곱의 노름.

$(q_1 q_2)^* = q_2^* q_1^*$

순서 뒤집힘! $(AB)^{-1} = B^{-1}A^{-1}$ 패턴.

단위 쿼터니언 = $S^3$

$$w^2 + x^2 + y^2 + z^2 = 1 \quad \longleftrightarrow \quad S^3 \text{ (4D 단위 구면)}$$

모든 3D 회전은 $S^3$ 위의 한 점. 이 구면 위에서 보간 = Slerp (p4_05).

축-각 → 쿼터니언

$$q = \cos\frac{\theta}{2} + \sin\frac{\theta}{2}(n_x i + n_y j + n_z k) = \left(\cos\frac{\theta}{2},\;\sin\frac{\theta}{2}\cdot\hat{n}\right)$$

왜 $\theta/2$인지는 p4_04에서 ($qvq^*$ 증명 후에). 지금은 변환 공식만.
