오일러 각 탐색기

Yaw(좌우) · Pitch(끄덕) · Roll(갸웃). 3D 좌표축이 세 번의 축 회전으로 어떻게 변하는지 확인해봐.

Yaw30°
Pitch0°
Roll0°

순서가 다르면 결과가 다르다

같은 Pitch=90°, Yaw=90°를 순서만 바꿔서 적용. 3D 회전은 비교환!

짐벌 락 — 자유도를 잃는 순간

Pitch를 90°로 올려봐. Yaw와 Roll이 같은 축을 돌리게 된다 — 자유도 하나를 잃는다.

Pitch=90°일 때 $R_x(90°)$가 y축과 z축을 정렬. Yaw든 Roll이든 같은 축을 중심으로 회전 → $\psi + \phi$만 의미. 독립 조절 불가.

Yaw0°
Pitch0°
Roll0°
짐벌 락! Yaw와 Roll이 같은 축을 돌린다

보간과 순서 문제

순서 의존성

$R_x(90°) R_y(90°) \neq R_y(90°) R_x(90°)$. 2D 회전은 교환 가능했지만 3D는 불가.

보간 불균일

오일러 각 Lerp: 경로가 곡선, 각속도 불균일, 짐벌 락 근처에서 튀김. Slerp 비교는 p4_05에서.

그래도 쓰는 이유

직관적, 3개 숫자로 저장, UI 슬라이더 친화. FPS 카메라는 Pitch ±85°로 제한하면 OK.
