Phase 1

동차 좌표 완전 이해

"Homogeneous"가 도대체 무슨 뜻인가

수학에서 "Homogeneous"란

이 단어가 헷갈리는 이유가 있다. 선형대수, 미분방정식, 사영기하학에서 전부 쓰이는데, 다 다른 맥락처럼 보인다. 하지만 공통된 핵심이 딱 하나 있다:

"스케일을 바꿔도 본질이 안 바뀐다"

동차 방정식 · $A\vec{x} = \vec{0}$

$\vec{x}$가 해면 $k\vec{x}$도 해. 스케일해도 해가 유지된다.

→ 해 공간이 스케일에 닫혀있음

동차 함수 · $f(t\vec{x}) = t^n f(\vec{x})$

입력을 스케일하면 출력도 예측 가능하게 스케일된다.

→ 스케일에 대해 "균질"하게 반응

동차 다항식 · 모든 항이 같은 차수

$ax^2 + bxy + cy^2$ — 모든 항이 2차. $ax^2 + bx + c$는 아님.

→ "같은 차수" = homo (같은) + genus (종류)

동차 좌표 · $(x, y, w) \sim (kx, ky, kw)$

전부 같은 비율로 스케일하면 같은 점. 비율만이 의미를 가진다.

→ 이것이 오늘 배울 것

왜 "동차"라는 이름이 붙었나

2D 직선의 방정식 $Ax + By + C = 0$이 있다. 여기에 $x = X/w$, $y = Y/w$를 대입하고 양변에 $w$를 곱으면:

$$AX + BY + Cw = 0$$

모든 항이 1차(degree 1). 방정식이 동차(homogeneous)가 되었다. 이 변환을 가능하게 하는 좌표 $(X, Y, w)$를 "동차 좌표"라고 부른다.

동차 다항식의 성질: $F(kX, kY, kw) = k^n \cdot F(X, Y, w)$. 원래 $F = 0$이었으면 $k^n \cdot 0 = 0$이므로, $(kX, kY, kw)$도 해다. 같은 비율로 스케일해도 해가 유지된다 — 이것이 동차 좌표의 이름의 이유이자 본질이다.

w가 뭔데? — 레이(ray)로 본다

2D 점 $(x, y)$를 동차 좌표로 쓰면 3D 공간의 점 $(X, Y, w)$가 된다. 실제 2D 점은 $w = 1$ 평면과의 교점으로 복원한다: $(X/w, \; Y/w)$.

핵심: 원점에서 $(X, Y, w)$를 잇는 직선(ray) 위의 모든 점은 같은 2D 점을 나타낸다. $(3, 2, 1)$과 $(6, 4, 2)$와 $(9, 6, 3)$은 3D에서는 다른 점이지만, 같은 ray 위에 있으므로 $w = 1$ 평면과 같은 곳에서 만난다 → 같은 2D 점 $(3, 2)$.

인터랙티브 — w를 바꿔보기

아래에서 $w$ 슬라이더를 움직이면, 동차 좌표 $(X, Y, w)$가 바뀐다. 하지만 $w = 1$ 평면 위의 실제 점은 움직이지 않는다.

w = 1.00

↑ 이 칩들은 전부 같은 2D 점을 나타낸다. 비율이 같으니까.

w = 0이면? — 방향 (무한 원점)

$w$를 0으로 슬라이더를 당겨봐. ray가 $w = 1$ 평면에 평행해진다. 교점이 없다 → 무한히 먼 점 → 방향(direction).

유클리드 기하에서 "평행선은 만나지 않는다"는 공리가 있다. 동차 좌표에서는 만난다 — $w = 0$인 무한 원점(point at infinity)에서. 사라지는 점(vanishing point)이 바로 이것이다.

1.1 연결 — w = 1이면 점, w = 0이면 방향

1.1에서 "원점을 옮기면 점의 좌표는 바뀌고 방향은 안 바뀐다"를 봤다. 그 이유가 이것이다: 이동 행렬을 곱하면 $w = 1$인 성분만 영향을 받고, $w = 0$은 무시된다.

왜 w = 1을 쓰는가? — 꼭 1이어야 하나?

아니다. $w$는 1이 아니어도 된다. $(6, 4, 2)$는 완벽히 유효한 동차 좌표이고, 점 $(3, 2)$를 나타낸다.

$w = 1$은 그냥 정규화된 대표값(canonical representative)이다. $X/w$, $Y/w$를 계산 없이 바로 읽을 수 있으니까 편한 것일 뿐.

실전에서 $w \neq 1$이 되는 순간이 있다:

원근 투영 후

카메라의 원근 투영 행렬을 곱하면 $w$에 깊이값($z$)이 들어간다. 가까운 물체는 $w$가 작고, 먼 물체는 $w$가 크다. $x/w$, $y/w$로 나누면 — 먼 것이 작아지는 원근감이 생긴다. 이것이 perspective divide.

가중 평균 (아핀 결합)

1.1에서 "위치 + 위치 = ???"이 의미 없다고 했다. 동차 좌표에서 $(x_1,y_1,1) + (x_2,y_2,1) = (x_1+x_2, y_1+y_2, 2)$. $w$로 나누면 $(\frac{x_1+x_2}{2}, \frac{y_1+y_2}{2})$ — 중점이다.

인터랙티브 — 이동 행렬의 비밀

$t_x$ 슬라이더로 이동량을 조절해봐. 점(w=1)은 움직이고, 방향(w=0)은 꿈쩍도 안 한다.

$t_x$ = 0.0

한 문장 정리

동차 좌표는 $w$라는 차원을 하나 추가해서,
N차원의 각 점을 (N+1)차원의 원점을 지나는 직선(ray)으로 표현하고,
이를 $w = 1$ 평면에 투영해 원래 점을 복원하는 체계다.
"동차"라는 이름은 이 좌표가 다항식 방정식을 동차(모든 항이 같은 차수)로 만들기 때문이다. 스케일 불변 — 이것이 수학 전체에서 "homogeneous"가 의미하는 단 하나의 본질이다.

직접 해보기

  1. 등가 확인: $w$를 2로 맞추고, 동차 좌표가 $(6, 4, 2)$인 걸 확인. 실제 점 $(3, 2)$는 안 바뀌는지.
  2. 방향 체험: $w$를 0으로 가져가봐. ray가 평행해지면서 "교점 없음 = 방향"이 되는 걸 확인.
  3. 이동 테스트: $t_x$를 움직여서 점만 이동하고 방향은 그대로인 걸 확인.
  4. 중점 계산: $(1, 0, 1) + (3, 0, 1) = (4, 0, 2)$. $w$로 나누면? 두 점의 중점인지 확인.
