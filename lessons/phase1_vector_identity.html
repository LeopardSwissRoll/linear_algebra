Phase 1

벡터의 이중 정체성

같은 숫자, 다른 의미

(3, 2)는 뭘까?

숫자 쌍 $(3, 2)$를 보자. 이건 두 가지로 읽을 수 있다.

위치(Point) — 공간에서 $(3, 2)$ 좌표에 있는 점.
방향(Direction) — 원점에서 $(3, 2)$으로 향하는 화살표.

데이터는 같다. 하지만 의미가 다르고, 다르게 행동한다. 아래에서 직접 움직여보면 무슨 말인지 보인다.

두 얼굴 — 같은 데이터, 다른 시각화

끝점을 드래그하세요

위치(Point) 해석

공간의 한 점. 좌표축에서 값을 읽어낸다.

방향(Direction) 해석

원점에서 뻗어나가는 화살표. 방향과 크기를 가진다.

이동하면 진짜 차이가 드러난다

두 해석이 진짜 다르다는 걸 보려면, 원점을 옮겨보면 된다.

위치(점)는 "어디에 있는가"이기 때문에, 기준점이 바뀌면 좌표가 바뀐다.
방향(화살표)은 "어디를 가리키는가"이기 때문에, 기준점을 옮겨도 좌표가 안 바뀐다.

아래에서 보라색 원점 O'를 드래그해봐.

이동 테스트

보라색 O'를 드래그하세요

위치 P (O' 기준)

← 원점이 바뀌면 좌표도 바뀐다

방향 d (O' 기준)

← 원점이 바뀌어도 좌표는 그대로

타입이 다르면 연산의 의미도 다르다

위치와 방향을 구분하면, 벡터 연산이 "의미"를 갖게 된다.

위치 − 위치 = 방향

두 점 사이의 변위.
$B - A$ = "A에서 B로의 화살표"

위치 + 방향 = 위치

점을 이동시킨다.
$A + \vec{d}$ = "A를 d만큼 옮긴 점"

방향 + 방향 = 방향

이동의 합성.
$\vec{d_1} + \vec{d_2}$ = "두 이동을 합친 이동"

위치 + 위치 = ???

수학적으로 정의되지 않음.
나중에: 가중 평균(아핀 결합)으로 의미 부여

Phase 3 예고 — 동차 좌표

이 구분은 나중에 동차 좌표(Homogeneous Coordinates)에서 정확히 공식화된다.

위치 = $(x,\; y,\; z,\; \mathbf{1})$ — $w = 1$
방향 = $(x,\; y,\; z,\; \mathbf{0})$ — $w = 0$
이동 행렬을 곱하면 $w=1$인 벡터만 영향을 받고, $w=0$은 그대로다. 바로 위에서 본 현상의 수학적 표현이다.

직접 해보기

  1. 두 얼굴: (0, 0)으로 드래그해봐. 위치와 방향 모두 "아무것도 없음"이 되는가?
  2. 이동 테스트: O'를 P와 같은 위치로 옮겨봐. P의 좌표가 (0, 0)이 되는 것을 확인.
  3. 방향 불변: O'를 아무리 움직여도 $\vec{d}$의 좌표가 안 바뀌는 것을 확인.
  4. 연산 체크: 위 캔버스에서, P의 좌표(O' 기준)는 $P - O'$와 같은지 계산해봐.
